Funcionalidades

Tudo que você precisa
no editor

Uma biblioteca completa para criação de modelos de documentos formais, pronta para integrar na sua aplicação React.

🖱️
Drag & Drop completo
Arraste, solte e redimensione campos livremente no canvas A4 com snap para grade configurável.
📐
Reflow por colunas
Campos de dados se organizam automaticamente em colunas, garantindo alinhamento profissional no documento.
🔍
Zoom e réguas
Zoom de 25% a 200% com réguas horizontais e verticais em milímetros para posicionamento preciso.
🖼️
Background customizável
Upload de imagem de fundo com biblioteca de backgrounds e toggle ativo/inativo em tempo real.
🏢
Header configurável
Header com logo, nome da empresa, subtítulo, gradiente de cor, opacidade e alinhamento personalizáveis.
↩️
Histórico de ações
Desfazer alterações com Ctrl+Z, copiar/colar e duplicar campos com atalhos de teclado.

Integração

Simples de integrar

Instale, importe e passe os callbacks. O editor cuida do resto.

App.jsx 
import { ModeloEditor } from '@torredeveloper-br/torresign'
import '@torredeveloper-br/torresign/dist/torresign.css'

export default function App() {
  return (
    <ModeloEditor
      onSalvar={(modelo) => api.salvarModelo(modelo)}
      onVoltar={() => navigate('/')}
      onUploadBackground={async (arquivo) => {
        const url = await storage.upload(arquivo)
        return url
      }}
      logoUrl="https://suaempresa.com/logo.png"
    />
  )
}
